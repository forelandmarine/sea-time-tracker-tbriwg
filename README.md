## About

SeaTime Tracker is a yacht sea time tracking application that automatically records days at sea by monitoring user-defined MMSIs via the MyShipTracking AIS data API. The app helps maritime professionals maintain compliant records for MCA sea service testimonials.

### Key Features
- Automatic sea time detection via AIS tracking
- Smart notifications for sea time confirmation
- MCA-compliant PDF and CSV report generation
- Vessel management and tracking
- Sea time logbook with detailed entries
